# Day22 演示用决策流程

```text
用户命令
  -> RGB视觉理解
  -> 第二组SafetyState（距离、闭合速度、TTC、有效性）
  -> Qwen高层理解
  -> 安全优先判断
  -> 高层动作
  -> A运行时
  -> B/C控制
  -> D最终安全仲裁
  -> CARLA执行与证据
```

红灯话术：
“驾驶员要求继续行驶。系统检测到前方红灯，安全约束优先，Qwen输出停车高层动作。车辆侧控制模块生成制动，D保留最终否决权。”

前车话术：
“LiDAR给出前车距离和TTC，Qwen只输出减速或目标速度，C负责纵向控制，D检查最终风险。”

行人话术：
“只有视觉类别和置信度可靠时，系统才使用行人原因；视觉无效时不会凭空声称存在行人。”

低置信度话术：
“感知状态不可靠时，系统输出停车并请求确认，不授权继续运动。”
