# Day21 多模态驾驶决策展示话术


## 场景1 红灯冲突

驾驶员:
继续走。


系统:
检测到前方红灯状态。


虽然驾驶员请求继续行驶，
但安全状态优先。


Qwen决策:
STOP。


原因:
交通规则和安全约束优先。
